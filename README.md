# 光伏體脂秤 UI 演示系统

一个交互式的光伏體脂秤用户界面演示工具，支持多种自动演示方案。

## 功能特性

- 🎨 完整的體脂秤UI界面模拟
- 🎬 多种自动演示方案（A、B、C、D）
- 📱 响应式设计，适配不同屏幕
- 🎯 精确的动画时序控制
- 🖼️ SVG点阵动画支持

## 使用方法

1. 打开 `index.html` 即可使用
2. 选择自动演示方案（A/B/C/D）
3. 点击"自动播放"查看演示流程
4. 或使用"上一步"/"下一步"手动控制

## 项目结构

```
├── index.html          # 主页面
├── 光伏秤UI.svg        # 主SVG显示文件
├── 121.svg            # 叠加图标
└── README.md          # 说明文档
```
