# 光伏秤UI - 部署说明

## 快速部署方法

### 方法1：Netlify Drop（最简单，无需注册）

1. 访问 https://app.netlify.com/drop
2. 将整个项目文件夹拖拽到页面上
3. 等待部署完成（约30秒）
4. 获得一个可分享的链接，例如：`https://xxxxx.netlify.app`

**优点**：无需注册，拖拽即用，自动生成HTTPS链接

---

### 方法2：GitHub Pages（免费，需要GitHub账号）

1. 在GitHub创建新仓库
2. 上传项目文件（index.html、121.svg、光伏秤UI.svg）
3. 在仓库设置中启用 GitHub Pages
4. 选择 main/master 分支
5. 获得链接：`https://你的用户名.github.io/仓库名`

**优点**：免费，可自定义域名，版本控制

---

### 方法3：Vercel（推荐，快速）

1. 访问 https://vercel.com
2. 使用GitHub账号登录
3. 点击 "New Project"
4. 导入GitHub仓库或直接上传文件夹
5. 自动部署，获得链接

**优点**：自动HTTPS，全球CDN，速度快

---

### 方法4：本地分享（临时）

如果需要临时分享给局域网内的人：

1. 使用Python快速启动本地服务器：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   ```

2. 访问：`http://localhost:8000`
3. 局域网内其他人访问：`http://你的IP:8000`

---

## 项目文件说明

部署时需要包含以下文件：
- ✅ `index.html` - 主页面
- ✅ `121.svg` - 叠加图标
- ✅ `光伏秤UI.svg` - 主SVG显示文件
- ❌ `光伏秤UI.ai` - 设计源文件（不需要部署）

---

## 注意事项

1. 确保所有SVG文件路径正确
2. 如果部署后图片不显示，检查文件路径大小写
3. 建议使用HTTPS链接分享（更安全）
